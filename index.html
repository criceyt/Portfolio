CFGS DAM

Desarrollo de una aplicación multicapa y multiplataforma para el registro y autenticación de usuarios en la base de datos de Odoo.

Organización: Grupos 3-4 personas | Duración: aprox 7 semanas (147h)

El Reto: El reto consiste en el desarrollo de parte de una aplicación multicapa y multiplataforma que permita el registro de usuarios en la base de datos de Odoo, así como su posterior identificación, autenticación y acceso a dicha aplicación. La planificación y seguimiento del proyecto se realizará con un software de gestión empresarial (ERP-CRM).

Descripción y requisitos:

  • Separación de la aplicación en capas de interfaz de usuario, lógica de negocio, acceso a datos y base de datos.
  • Uso de un repositorio público online de código fuente con sistema de control de versiones (GIT).
  • Programación de servicios y procesos: aplicación cliente/servidor con comunicación mediante sockets TCP.
  • Acceso a Datos: análisis y estudio de las tablas, campos y relaciones en Odoo.
  • Desarrollo de Interfaces: interfaz de usuario de tipo escritorio desarrollada con JavaFX y FXML.
  • Documentación del código mediante Javadoc.
